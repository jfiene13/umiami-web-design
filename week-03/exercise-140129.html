Building Your First Page

You have learned to create a basic HTML page, tags to format text and structure a document, lists and tables to format data and prepare images for the web.

Let's put all those tools to use.

Your assignment is to create a one-page document with information about the 2012 Presidential Election. We've provided most of the content (see election.txt), but you are welcome to add more.

Requirements

  1. A proper HTML document foundation (i.e. head, body, etc.)
  2. Content organized using the array of structural tags (i.e. header, section, article, etc.)
  3. Textual content about the election (provided)
  4. A data table (provided)
  5. An image of the electoral map (provided)
  6. Lists of the top 5 states for each candidate (provided)

A few notes...

• Your data table is presented in csv (comma separated value) format. It can be organized horizontally, vertically. Do you find one way is easier to read or understand than another? Experiment!

• You will want to re-size the electoral map image to a reasonable height/width and save it in a proper format. Technically this graphic is a figure. Is there a way to mark up a figure and give it a proper caption in HTML?

• Be creative! Perhaps you would like to include more information such as a brief bio and/or photo of each candidate in a separate article. Maybe you come across some additional statistics that can be presented as a table and/or list. What about links to web sites about the election? Maybe you can add a video?

• Be sure to keep your files organized (i.e. images in the img directory).

• Only use content which is public domain and/or licensed through Creative Commons. Always remember to cite your sources!
