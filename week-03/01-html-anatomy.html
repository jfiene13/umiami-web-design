HTML Anatomy

This is an example of a set of tags. Tags are elements. (Most) tags have an opening and closing element.

<element></element>

Some tags contain additional information. We provide this through the use of attributes and values.

<element attribute="value"></element>

Example 1

<p class="intro">This is a sentence.</p>

  1. What is the element?
  2. What is the attribute?
  3. What is the value?
