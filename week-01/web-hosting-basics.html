Web Hosting Basics

Web Host

  • a computer dedicated to serving a web site
  • storage for files
  • processing power for scripts
  • bandwidth for delivering content

Additional Services

  • email
  • databases
  • site security
  • domain management

Types of Hosting

  • Shared (multiple users, ready to go)
  • Virtual Private Server (VPS) (custom software)
  • Dedicated (custom hardware)
  • Cloud (flexibility)

Domain Names

  • your 'address' on the web
  • points to an Internet Protocol (IP) address
    • a numerical label assigned to an Internet-connected device (129.171.233.80)
    • What is your IP address? (findmebyip.com)

Anatomy of a domain

http://www.apple.com

  • http:// - Hypertext Transfer Protocol
  • com - top-level domain
  • apple - domain
  • www - subdomain

Other Top-Level Domains (TLDs)

  • generic: .com, .net, .org, .edu, .gov, .mil
  • country: .us, .ca, .de, .ru, .se

Finding a Domain

  • http://domai.nr

Domain Name Tips

  • make it easy to remember
  • keep it short (8 characters is great!)
  • try for a .com, but other TLDs are okay
  • avoid hyphens, numbers
  • avoid copyright infringement
