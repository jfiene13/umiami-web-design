Web Graphics

Types of Graphics

Raster

  • grid of pixels
  • cannot scale up

Vector

  • points, lines, curves, shapes
  • can be scaled up

Compression

  • necesssary for images, multimedia
  • data encoding disgards information
  • allows file size to be smaller
  • compression » data loss » loss of quality » smaller file size

Lossy vs. Lossless

  • lossy: file loses data in compression
  • lossless: no data loss in compression

Formats

GIF: Graphics Interchange Format

  • .gif
  • limited to a palette of 256 colors
  • good for low-color, graphics such as logos
  • small animations, transparency

PNG: Portable Network Graphics

  • .png
  • a replacement for GIFs (PNG-8)
  • color palate, transparency options (PNG-24)
  • supported in all OSes, modern browsers
  • smoother (anti-aliased) graphics, no animation

JPEG: Joint Photographic Experts Group

  • .jpg
  • millions (billions?) of colors
  • used for photography
  • compression may leave 'artifacts'
  • progressive: photo displays portions of image over time
  • maximum size: 65535 pixels by 65535 pixels

SVG: Scalable Vector Graphics

  • .svg
  • vector image format
  • behaviors described in XML
  • supported by all major browsers since 2011
  • not yet fully adopted

Adobe Photoshop

  • used to edit raster images
  • File » Save for Web & Devices
  • allows you to save a raster image in raster format (GIF, JPEG, PNG)

Adobe Illustrator

  • used to edit vector images
  • File » Save for Web
  • allows you to save a vector image in raster format (GIF, JPEG, PNG)
  • also allows you to save in vector format (SVG)

Tips

  1. Always retain your original images.
    • once an image is compressed, its data is lost
  2. Work in RGB format.
  3. Check your resolution: 72dpi.
    • resolution settings are in the Image Size panel
  4. Photos should always be a JPEG.
  5. Graphics should always be a GIF or PNG.
    • preferably a PNG, unless you must support old browsers
  6. Size down, never up.
    • vector images are the only exception
  7. Do not distort your images with unbalanced proportions.
    • automatically taken care of in Photoshop (Image Size)
    • make sure "Constrain Proportions" is selected
  8. Always consider size vs. quality
    • Does your design focus on strong imagery?
    • What considerations are you making for mobile?
